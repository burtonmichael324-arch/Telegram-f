Telegram Forwarder Dashboard (Online) - Railway

SETUP:
1. Sign up at https://railway.app/
2. Create a new project -> Deploy from ZIP
3. Upload this ZIP file
4. Railway will assign a URL (HTTP)
5. Open the URL to access your dashboard
6. Add source channels and click Start

NOTE:
- Your Telegram account must be joined to the source channels
- First run will ask for Telegram login code
- Bot runs 24/7 as long as Railway project is running
